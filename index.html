Hyperaccurate Pathtracer
This system was made to be a very accurate pathtracer, or what most people would typically call raytracing.
It can calculate everything from: Lighting, Textures, PBR, Reflections, Refractions, Caustics, HDRI (Sky), and more.
This would not exist without PoptartNoah's pathtracer module, which I heavily modified and optimized to make this system possible.
Nearly no original code from PoptartNoah's module exists in this system, however it would be wrong to say that it's a completely original creation.
